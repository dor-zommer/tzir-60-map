# ציר 60 — מפת השילוט

מפה אינטראקטיבית לתחקיר על מונופול השילוט של החתן של אורית סטרוק.

## מבנה
- `index.html` — המפה (קובץ עצמאי אחד)
- `fonts/` — פונטי Next Exit
- `images/` — תמונות הכתבה (הוסף לכאן)

## איך לערוך תוכן

פתח `index.html` בעורך טקסט. גלול עד `const CONTENT = {`. לכל נקודה:

```js
"צומת תפוח": {
  image: "images/tapuach-sign.jpg",
  caption: "השילוט בצומת תפוח, אפריל 2026",
  text: [
    "פסקה ראשונה של הטקסט.",
    "פסקה שנייה.",
    "פסקה שלישית."
  ],
  credit: "צילום: סיון תהל | תחקיר: דור זומר"
}
```

- `image` — שם קובץ ב-`images/` או URL חיצוני. השאר `""` לפלייסהולדר.
- `text` — מערך של פסקאות. כל מחרוזת = פסקה נפרדת.
- אל תשנה את שמות המפתחות (שמות הנקודות) — הם חייבים להתאים ל-KML.

סמן אדום פועם = נקודה לתיעוד. סמן ענבר = נקודה שמילאת.

## הטמעה באתר

### אפשרות 1: העלאה לשרת והטמעה ב-iframe
העלה את כל התיקייה לשרת (למשל `ha-makom.co.il/maps/tzir-60/`), ואז:

```html
<iframe src="https://ha-makom.co.il/maps/tzir-60/"
        width="100%" height="720"
        style="border:0" loading="lazy"></iframe>
```

המפה מזהה אוטומטית שהיא ב-iframe ועוברת למצב קומפקטי (כותרת קטנה יותר, בלי חתימת "המקום הכי חם בגיהנום" — היא כבר בדף הכתבה מסביב). ה-iframe שולט בגובה.

### אפשרות 2: פרסום עצמאי
העלה כמו שהוא — יש URL נפרד עם כותרת ו-meta.

## הוספת/הסרת נקודות

אם מתווספות נקודות ל-KML, ערוך גם את `POINTS` (קואורדינטות) וגם את `CONTENT` (מפתח ריק). שני המערכים חייבים להכיל את אותם שמות.
